## 🛠️ Tech Stack & Local Execution

* **Frontend**: React 19 + TypeScript + Tailwind CSS 4.0 (for fluid high-contrast layouts)
* **Animation Engine**: `motion/react` (for micro-interactions and route fade-ins)
* **Backend Runtime**: Express Proxy + Vite node middleware (ensures all Stripe, API or database layers remain securely private and server-side)
* **PDF Utility**: `jsPDF` for client-side generation of beautiful certified Golden Covenant pledges.

### Scripts
* **Install Dependencies**:
  ```bash
  npm install
  ```
* **Run in Dev Mode** (starts the Express + Vite server on standard port `3000`):
  ```bash
  npm run dev
  ```
* **Build App for Production** (bundles client chunks and esbuilds the server into CommonJS structure):
  ```bash
  npm run build
  ```
* **Start Production Server**:
  ```bash
  npm run start
  ```
